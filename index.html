DashCat

The prettiest way to browse GitHub. Get notifications, merge pull requests and discover interesting projects!.

Download

Merge all the things

GitHub changed the way people work with each other, so DashCat stands on a giant's shoulder. Fast and quick ways to diff files, check merge status and merge any given pull request.

Notifications

Getting that pull request and knowing your pending notifications it's really useful but going to your browser will end up browsing Facebook or Twitter. DashCat send system notifications with all the relevant info and you can check it without losing focus.

i18n

Code is for everyone, even if you don't speak english so the app can be easily translated in several languages without having to wait for GitHub itself.

Native osx app. Kind of.

HTML5 it's ready for heavylifting so insted of write an app per systems it's just a webapp wrapped in a Cocoa Application so it's pretty, fast and easily editable.

Open Source. Free as beer.

GitHub is the open source heaven, DashCat is a GitHub viewer then DashCat must be open source!. The only way to make this a really useful app is having great people contributing.

Designed to fill a gap. Make everyone happy.

We <3 GitHub but we all know that it won't make everyone happy everyone wants different features, styles and functionalities. DashCat it's pure html so everyone can edit it, just clone it edit it and use it as you wish.
